* 클래스 설계 예시 *
동물병원 --> 초코(푸들), 삐삐(고양이), 하양이(토끼)
클래스 설계 : Dog, Cat, Rabbit
            Dog(자식) extends Animal :: 부모 클래스와 자식 클래스를 연결하는 키워드
            속성 : 이름, 혈액형, 색깔
            행위 : 먹는다, 잔다
            -> 부모의 속성과 행위를 상속 받아 사용할 수 있다.

            Cat
            속성 : 이름, 혈액형, 색깔
            행위 : 먹는다, 잔다

            Rabbit
            속성 : 이름, 혈액형, 색깔
            행위 : 먹는다, 잔다

        => Dog, Cat, Rabbit의 특성이 비슷하므로 그 특성을 모아 부모 클래스 생성
            Animal(부모)
            속성 : 이름, 혈액형, 색깔
            행위 : 먹는다, 잔다

Dog 초코(객체) = new Dog("초코", "A", "Brown");
Cat 삐삐 = new Cat("삐삐", "B", "White");
Rabbit 하양이 = new Rabbit("하양이", "A", "Gray");
